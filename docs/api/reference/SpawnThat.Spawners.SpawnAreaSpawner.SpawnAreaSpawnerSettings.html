Class SpawnAreaSpawnerSettings

SpawnArea spawner builder settings. Any property not set will be ignored.

Any setting not set for a builder will mean Spawn That will use the existing setting when overriding a template, or set a default value if creating a new.

Intended as an optional way to configure ISpawnAreaSpawnerBuilder.
