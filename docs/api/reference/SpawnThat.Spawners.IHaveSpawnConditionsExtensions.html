SetConditionAltitude<T>(T, Nullable<Single>, Nullable<Single>)

Set allowed altitude range in which spawner is active.

Altitude is calculated as the spawners vertical distance to water-level. Meaning altitude 0 is at the water surface. If negative, it means below water, positive means above.

Note: Depending on spawner type/settings, spawning can still be done at other altitudes. Use to control the allowed spawn positions.

SetConditionAreaSpawnChance(ISpawnAreaSpawnBuilder, Single)

Set spawn chance based on each area.

Each area rolls chance pr template id once pr seed. If the chance roll is less than listed here, this area will never activate this template, and vice versa.

This allows for sitations where only some areas (eg., 10% of blackforests) will have a spawn show up.

SetConditionOceanDepth<T>(T, Nullable<Single>, Nullable<Single>)

Set allowed ocean depth range in which spawning is active.

Ocean depth is defined as distance to water surface. And is calculated as:

Max(0, WaterLevel - Seafloor)

This means the water-surface and above will be ocean depth 0. An ocean floor at 10 units below the surface will be 10.

SetConditionTilt<T>(T, Nullable<Single>, Nullable<Single>)

Set tilt range of terrain, for which spawning is active.

Tilt is calculated as 0 when perfectly flat, and 90 when completely vertical.

Note: Valheim calculates tilt extremely inconsistently throughout the codebase. Tilt above 45 needs to be tested before being relied upon, as there is at least one buggy calculation in vanilla.
