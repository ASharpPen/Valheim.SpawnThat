Class ModifierDefaultRollLevel

Simulates the vanilla level assignment logic.

Level is assigned by setting level to Min(1, minLevel).

If minDistanceForLevelups is greater than distance to center, and maxLevel is greater than level, the levelupChance is rolled. This is repeated until either maxLevel is reached, or a levelup roll fails.
